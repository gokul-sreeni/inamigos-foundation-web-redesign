A modern UI/UX redesign and website improvement analysis created for **InAmigos Foundation** as part of an internship task focused on improving accessibility, emotional storytelling, donation experience, mobile responsiveness, and overall user engagement.

---

## 📌 Project Overview

This project analyzes the existing InAmigos Foundation website and proposes a complete modern redesign focused on:

- Better user experience
- Improved accessibility
- Stronger visual hierarchy
- Emotional storytelling
- Trust-building donation flow
- Mobile-first responsiveness
- Cleaner and modern NGO branding

The redesign was created to demonstrate practical UI/UX thinking and frontend presentation skills while maintaining the humanitarian purpose of the organization.

---

## 🎯 Objectives

- Analyze the current website experience
- Identify UI/UX and accessibility issues
- Suggest realistic design improvements
- Redesign key sections of the website
- Improve engagement and donation conversion
- Create a professional internship submission
